# 🌾 AgriPulse - Precision Agriculture AI

AgriPulse is an **AI-powered web application** that analyzes soil nutrient data (N, P, K, pH) and predicts **crop suitability**, identifies deficiencies, and provides **fertilizer recommendations** with farmer-friendly explanations. Built for HackArena 2026, it helps small-scale farmers **make data-driven planting and fertilization decisions**.
